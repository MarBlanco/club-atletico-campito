# CLUB ATLÉTICO CAMPITO

## Purpose

Sitio web institucional y CMS del Club Atlético Campito.

## Mission

Representar digitalmente la identidad, historia, equipos y comunidad del club.

## Core Principles

- Simplicidad
- Comunidad
- Formación
- Fútbol
- Identidad institucional

## Audience

- Jugadores
- Familias
- Hinchas
- Dirigentes
- Comunidad local

## Not

- No es una red social
- No es un sistema complejo
- No es una plataforma de membresías
- No requiere registro de usuarios públicos

## Official Roles

### Admin

Acceso total al sistema.

### Colaborador

Gestión de contenido.

## Approved Public Sections

- Home
- Club
- Equipos
- Primera
- Noticias
- Fixture
- Multimedia
- Momentos Campito
- Contacto

## Official Logo

Usar exclusivamente el escudo oficial CAC aprobado por el club.

No reemplazar.
No rediseñar.
No reinterpretar.

## Design Philosophy

Institucional.
Deportivo.
Claro.
Visual.
Simple.
