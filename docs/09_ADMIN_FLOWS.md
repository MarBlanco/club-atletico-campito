# ADMIN FLOWS

# ROLES

## Admin

Full Access

Can:

- manage users
- manage content
- manage settings
- manage structure
- manage media

---

## Collaborator

Content Management Only

Can:

- create content
- edit content
- upload media

Cannot:

- manage users
- manage roles
- modify system settings

---

# LOGIN FLOW

User enters:

- email
- password

System validates credentials.

Redirect to dashboard.

---

# DASHBOARD FLOW

Displays:

- recent news
- upcoming matches
- latest galleries
- quick actions

Quick Actions:

- New News
- New Player
- New Match
- New Gallery

---

# NEWS FLOW

Create News

1. Open News
2. Click New
3. Fill fields
4. Upload image
5. Publish

Edit News

1. Open News
2. Select item
3. Edit
4. Save

---

# PLAYER FLOW

Create Player

1. Open Players
2. New Player
3. Upload photo
4. Complete data
5. Save

Edit Player

1. Select player
2. Update data
3. Save

---

# STAFF FLOW

Create Staff Member

1. Open People
2. Select category
3. Upload image
4. Complete fields
5. Save

---

# MATCH FLOW

Create Match

1. Open Fixture
2. New Match
3. Fill rival
4. Select date
5. Save

Update Result

2. Enter result
3. Save

---

# GALLERY FLOW

Create Gallery

1. Open Momentos Campito
2. New Gallery
3. Select category
4. Select match date
5. Upload cover image
6. Save

---

# MEDIA FLOW

Upload Photos

1. Select gallery
2. Upload files
3. Preview
4. Crop
5. Save

Upload Videos

1. Select gallery
2. Upload video
3. Generate thumbnail
4. Save

---

# MOMENTOS CAMPITO FLOW

Weekly Process

Community Manager receives:

- Primera photos
- Infanto photos
- Femenino photos
- Veteranos photos
- Families photos
- Fans photos

Workflow:

1. Create gallery
2. Select category
3. Upload content
4. Publish

System automatically displays content in:

- Momentos Campito
- Multimedia

If gallery is linked to a match:

- Match Gallery

---

# CLUB FLOW

3. Save

# USER MANAGEMENT FLOW

Admin Only

4. Save

3. Save
